# Xsuite tutorial (seminar)

This folder contains a small, self-contained set of Jupyter notebooks used for an Xsuite tutorial.

## Notebooks

The notebooks are meant to be run in order:

- `notebook_00_lattice_design.ipynb` – lattice/model setup
- `notebook_01_optics_matching.ipynb` – basic matching workflows
- `notebook_02_slow_extraction.ipynb` – slow extraction example (PIMM)
- `notebook_03_rf_and_impedance.ipynb` – RF/impedance example
- `notebook_04_lhc_lossmap.ipynb` – LHC loss map example
- `notebook_05_formats.ipynb` – common file formats used in the workflow

## Running

From this directory:

```bash
jupyter notebook
```
